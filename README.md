# dent

An opinionated code formatter for NSIS scripts. This monorepo contains both, the separate library and CLI packages:

- [library](packages/dent)
- [CLI](packages/dent-cli)

The library not only powers the CLI, but is also used to format NSIS code in [Visual Studio Code](https://github.com/idleberg/vscode-nsis) and [Pulsar](https://github.com/idleberg/atom-language-nsis).
