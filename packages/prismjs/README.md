# @nsis/prismjs

![License](https://img.shields.io/npm/l/@nsis%2Fprismjs?style=for-the-badge)
[![Version](https://img.shields.io/npm/v/@nsis/prismjs?style=for-the-badge)](https://www.npmjs.org/package/@nsis/prismjs)
[![Build](https://img.shields.io/github/actions/workflow/status/idleberg/nsis-org/ci.yml?style=for-the-badge)](https://github.com/idleberg/nsis-org/actions)

NSIS syntax highlighter for [PrismJS](https://github.com/PrismJS/prism), a lightweight, extensible syntax highlighter, built with modern web standards in mind.

[Demo Time](https://idleberg.github.io/nsis-org/prismjs/) 🙌

## Installation

```bash
# Install
$ npm install @nsis/prismjs
```

## Usage

```typescript
import Prism from "prismjs";
import "@nsis/prismjs";

Prism.highlightAll();
```
